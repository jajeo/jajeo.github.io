mongoose文档翻译 － Schemas（二）

2016/1/5 posted in  Node.js

接着之前第一篇继续了

选项

Schemas有少许可配置的选项可以直接传递给构造函数或者set函数：

new Schema({...}, options);

//或者

var schema = new Schema({...});
schema.set(option, value);

可用的选项：

  • autoIndex
  • capped
  • collection
  • emitIndexErrors
  • id
  • _id
  • minimize
  • read
  • safe
  • shardKey
  • strict
  • toJSON
  • toObject
  • typeKey
  • validateBeforeSave
  • versionKey
  • skipVersioning
  • timestamps
options: autoIndex

在程序开始的时候，Mongoose为在Schema中定义的每一个索引发送一个ensureIndex的命令。在Mongoose v3版本中，indexes在backgound中默认创建。如果你想要禁用自动创建的功能并且你想在索引创建的时候手动处理，设置Schema的autoIndex选项为false，并且在model中使用ensureIndexes方法。

var schema = new Schema({...}, {autoIndex: false});
var Clock = mongoose.model('Clock', schema);
Clock.ensureIndexes(callback);
